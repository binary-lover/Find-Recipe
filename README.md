# React + Vite


## Project run
```
git clone https://github.com/binary-lover/Find-Recipie
```

## setup .env file
```
VITE_APP_ID=YOUR_APP_ID
VITE_APP_KEY=YOUR_APP_KEY
```

## Install dependencies
```
npm install
```

## Run the project
```
npm run dev
```
